# cnctd_go

Go toolchain wrapper for programmatic builds from Rust.

## Installation

```toml
[dependencies]
cnctd_go = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- Build Go projects
- Manage modules
- Cross-compilation support
